Control del robot ESP32

Panel limpio para controlar el auto por HTTP o WebSocket. Para el código ESP32 que venimos usando, selecciona HTTP nativo y envía comandos tipo /control?cmd=w.

Último comando: -- Modo: HTTP Seguridad: Activo

Control principal

Usa los botones o el teclado. Mantén el control simple: avanzar, retroceder, girar y detener.

Teclas: W avanzar, A izquierda, S retroceder, D derecha, X detener, Q/E servo.

Telemetría

En modo HTTP, el ESP32 actual solo responde comandos. La telemetría completa la publica por MQTT. En modo WebSocket, se actualizará si tu ESP32 envía JSON o mensajes tipo DIST:25.

Distancia -- cm
Estado Sin datos sensor
Motor derecho -- valor
Motor izquierdo -- valor
Servo -- grados
Última actualización -- hora
Con tu código ESP32 actual, esta página controla por HTTP. Para ver telemetría en el navegador necesitas agregar un endpoint HTTP de sensores, un WebSocket en el ESP32 o MQTT sobre WebSocket en Mosquitto.
