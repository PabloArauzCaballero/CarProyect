Control Robot ESP32

Panel para controlar el auto usando el ESP32 como puente hacia el Arduino Nano. En modo HTTP se envían comandos a /control?cmd= y se leen sensores desde /telemetry.

Desconectado Último comando: -- Modo: HTTP

Recomendado: primero prueba en el navegador http://IP_DEL_ESP32/telemetry. Si eso no devuelve JSON, la web no podrá mostrar datos.

Control principal

Usa los botones o el teclado. W avanzar, A izquierda, S retroceder, D derecha, X detener, Q/E mover servo. Direcciones quedan activas hasta presionar STOP.

Si el ESP32 dice que envía comandos pero el Nano no se mueve, revisa baudrate, GND común, TX/RX cruzados y alimentación de motores.

Telemetría

Esta vista consulta /telemetry cada 700 ms. El ESP32 debe responder JSON: {"motor_der":0,"motor_izq":0,"servo":90,"distancia":7}.
